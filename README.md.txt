# 🏢 DOUKÈ Compta Pro

## 📋 Description
Application de gestion comptable conforme au système SYSCOHADA Révisé, spécialement conçue pour les entreprises en Côte d'Ivoire.

## ✨ Fonctionnalités
- 📊 Tableau de bord comptable
- 📝 Saisie d'écritures comptables
- 📈 Suivi des comptes clients et fournisseurs
- 🧾 Gestion de la TVA
- 📋 États financiers
- 🔄 Rapports de gestion

## 🚀 Déploiement
Cette application est déployée sur Render.com

## 📱 Technologies
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## 👨‍💻 Auteur
Développé avec ❤️ pour la comptabilité de l'Afrique de l'ouest
